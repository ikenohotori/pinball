# Orbital Cadet Pinball

スマホ縦向けブラウザ向けの Phaser 3 + Matter ベースのピンボールMVPです。Windows版3Dピンボールの操作感を参考にしつつ、レイアウトと演出は新規構成にしています。

## 開発

```bash
npm install
npm run dev
```

## 操作

- 画面左下: 左フリッパー
- 画面中央下: プランジャー長押し、離して発射
- 画面右下: 右フリッパー
- キーボード: Left / Right / Down または Space

## 実装済みMVP

- タイトル、ゲーム、本体、ゲームオーバーのシーン遷移
- Matter 物理によるボール、バンパー、ターゲット、排出口
- 左右フリッパーとプランジャー入力
- スコア、一球勝負のゲームオーバー、簡易ミッション、ハイスコア保存
- 合成音による簡易SE
